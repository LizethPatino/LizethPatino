## What I'm building

### [Vibemosphere](https://github.com/LizethPatino/vibemosphere)
*An AI-powered emotional mirror for illustrators.*

Vibemosphere reads the emotional vibe of an illustration and reflects it back — as a poetic title, mood tags, a song recommendation, and a gentle reflection — so the creator can recognize what they were feeling all along.

**The AI doesn't interpret for you. It holds up a mirror.**

Built with React 19 · TypeScript · Gemini 2.5 Flash · Express · Supabase

→ [View project](https://github.com/LizethPatino/vibemosphere)
